## ⚙️ Pre-Installation
Before installing CyberVision, complete the following steps:

### **Required Utilities**
- **.NET SDK** :  
  - [.NET 8.0 SDK - Windows x64 Installer](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)  
  - [.NET 9.0 SDK - Windows x64 Installer](https://dotnet.microsoft.com/en-us/download/dotnet/9.0)
- **Microsoft Visual C++ Redistributable** :  
  - [x86](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170)  
  - [x64](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170)

### **Official Game Content**
- Make sure your game is clean and updated  
- Launch once to initialize REDmod ( Steam users need to download Cyberpunk 2077 REDmod before )

### **Steam Overlay**
- Disable Steam overlay globally or just for Cyberpunk to avoid crashes:  
  1. Steam → Settings → In-Game → Uncheck “Enable Steam Overlay”  
  2. OR Library → Cyberpunk 2077 → Right click → Properties → Uncheck “Enable Steam Overlay”  

### **Installation Location**
- **Do NOT install in C:\Program Files** or C:\Program Files (x86)  
- Recommended: `C:\Games\Cyberpunk` or `D:\Modlist\CyberVision`  

### **Antivirus**
- Disable or remove 3rd-party antivirus (Webroot, Bitdefender)  
- Add **Wabbajack list folder** to antivirus exclusion  

### **Pagefile / Virtual Memory**
Larger modlists require significant memory. Regardless of RAM/VRAM, this step is **required**:  
1. Press **Win + R** → `sysdm.cpl ,3` → Performance → Advanced → Virtual Memory → Change  
2. Uncheck "Automatically manage"  
3. Select fastest SSD → Custom size: `40960 MB` initial and max → Set → Apply → Restart  

### **NVIDIA Users - Shader Cache**
- NVIDIA Control Panel → Manage 3D Settings → Shader Cache Size → Set to **10 GB** → Apply

## 🛠 Installation
1. Delete all Cyberpunk data  
2. Launch once to activate REDmod  
3. Install CyberVision via Wabbajack  
4. Select MO2 profile (12 profiles: 6 English, 6 French)  

For **setup instructions, configuration details, and troubleshooting**, check the **CyberVision Discord section**: [Discord Link](https://discord.com/invite/WSFgf5S7wS)
